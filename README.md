# ScreenShotTool
最近摸鱼严重，练练手模仿微信的截图功能做的一个小玩意儿。
 
### 已实现：
1.支持截图时隐藏界面  
2.支持双击、点击保存按钮保存到指定文件夹  
3.支持移动截图区域，支持改变截图区域  
4.支持放大镜，支持展示鼠标位置及RGB值  
  
### 未实现计划中：
1.截图区的绘制功能，方框，椭圆，箭头，线条，文字，图片  
2.绘制内容的移动和修改功能  
3.绘制操作的撤回功能
